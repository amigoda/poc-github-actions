API Endpoints

  • /api/time
    • Returns the current time of the server.
    • Format:
      ⇒ /api/time?timeZone=<timezone>
    • Parameters:
      • Without params it returns the time in the default time zone.
      • timeZone ⇒ The time zone. Examples: GMT, UTC, GMT+2, UTC-5, ...
    • Examples:
      • /api/time
      • /api/time?timeZone=UTC
      • /api/time?timeZone=UTC+1
      • /api/time?timeZone=UTC-1
      • /api/time?timeZone=GMT
      • /api/time?timeZone=Europe/Paris
